### npm安装使用

```bash
npm install vve-rjucd-iconfont
```

引入样式
```bash
import 'vve-rjucd-iconfont/dist/vve-rjucd-iconfont.css'
```

如果你的项目支持sass，你可以直接引用
```bash
import 'vve-rjucd-iconfont/src/index.scss'
```

### CDN安装使用

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/vve-rjucd-iconfont@1.1.1/dist/vve-rjucd-iconfont.css">
```

## 开发

- 安装依赖

```bash
npm install
```

- 开发

```bash
npm start
```

- 打包

```bash
npm run build
```

- 发布

```bash
npm run release
git push --follow-tags origin master && npm publish
```
